4. Installation des Dépendances
Avant de lancer le serveur, assurez-vous d’installer Flask et les dépendances :

sh
Copier
Modifier
pip install -r requirements.txt
Si requirements.txt est vide, vous pouvez l'ajouter avec :

makefile
Copier
Modifier

Puis exécutez :

sh
Copier
Modifier
